Welcome to the MyJS MVC project site

As the name probably suggests you, MyJS MVC is a model, view, controller based application framework written entierly in JavaScript.
This a tipical "i didn't find any fitting solution so I created my own" story.

When I stared to design the framework my goals were:

  • Small application footprint
  • Library independent
  • Modular system (Lazy loading, AJAX support)
  • Understandable structure
  • Familiar syntax
As we know "seeing is believing", so here is a Hello world example: 

    // Initialize MyJS MVC
    app.model = new Model();
    app.view = new View( app.model );
    app.controller = new Controller();  

    // Create model (it can be an array or an object as well)
    app.model.set( 'header', 'Hello world!' );

    // Create a view (add rendering logic)
    app.view.register('list', function( data )
    {
        var tpl = '
' + data.header + '
';
        return tpl;
    });

    // Add a controller
    app.controller.register( 'hello_world', function()
    {
        // Connect the model to the template
        app.view.assign( 'list', 'header' );

        // Render the template into the element with the id: "dst" 
        app.view.render( 'list', 'dst' ); 
    });
	
	// Attach the controller to a button with an event
    app.controller.toElement( $('my_button'), 'click', 'hello_world' );
